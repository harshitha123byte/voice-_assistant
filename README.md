# 🎙️ AI Voice Assistant Using Python

An AI-powered Voice Assistant developed using Python that listens to voice commands and performs various tasks like searching Wikipedia, opening websites, telling the time and date, playing YouTube videos, opening applications, telling jokes, and much more.

## Features

- Speech Recognition
- Text-to-Speech
- Google Search
- Wikipedia Search
- Play YouTube Videos
- Tell Current Time
- Tell Current Date
- Open Calculator
- Open Notepad
- Tell Jokes
- Weather Information (Optional)

## Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- PyWhatKit
- Webbrowser
- OS Module
- Datetime

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Voice-Assistant.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python voice_assistant.py
```
